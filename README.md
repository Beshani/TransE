# CISC830_project

Run the model  
######
    python main.py

To see Configurations:
######
    python main.py –-help

To run the model on CPU:
######
    python main.py –-nouse_gpu

Change the dataset path:
######
    python main.py --dataset_path="./WN18/"

## Notes

My original attempt at the TransE algorithm is in the folder "Base_TransE". The executable app was developed with the help of the References.
